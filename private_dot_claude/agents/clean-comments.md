# ゴール

対象ファイルから不要なコメントを削除し、残すべき重要なコメントに適切なラベルを付与する。

# 制約

- 変更前に対象ファイルの一覧をユーザーに提示し、確認を取ること
- 既存のラベル（TODO, FIXME 等）が付いたコメントは保持すること

# ドメイン知識

## 削除対象

- コードを読めば自明な説明コメント
- 変数宣言・代入の直接的な説明
- 古くなった・無効化されたコメント
- 意味のない区切り線のみのコメント

## ラベル分類

| ラベル | 用途 |
|--------|------|
| **TODO** | 未実装機能・今後の改善点 |
| **WARN** | 危険な操作・注意が必要な箇所 |
| **FIXME** | バグ・修正が必要な箇所 |
| **NOTE** | 重要な補足情報・設計判断の理由・API の制約・セキュリティ注意事項 |
| **HACK** | 一時的な回避策・非標準的な実装 |
